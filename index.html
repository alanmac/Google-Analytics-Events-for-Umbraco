Umbraco Google Analytics Events Tracking

A package for adding manageable GA Event Tracking to your Umbraco site.

Install

  1. Install the Umbraco-GA-Event-Tracking package
  2. Include the event tracking script via the Html Helper anywhere AFTER you have included Google Analytics in your master template page.

Including Google Analytics

For your convenience, we have a handy helper available for outputting the GA script. You do not have to use this method if you have already included GA but if you do, make sure to set your API key.

Set your API key in the properties of the Event Tracking content node

@Html.IncludeGoogleAnalytics()

Including the Tracking Script

As above, just add this bit of razor code anywhere after you include GA in your master template

@Html.IncludeGaEventTracking()
